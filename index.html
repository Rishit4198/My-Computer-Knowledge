HTML

  • HTML stands for HyperText Markup Language.
  • It is used to create the structure of web pages.
  • HTML elements are represented by tags.
  • Common tags include <html>, <head>, <body>, <p>, <a>, etc.
  • HTML documents are saved with a .html or .htm extension.
  • HTML can be styled with CSS and made interactive with JavaScript.

Python

  • Python is a high-level, interpreted programming language.
  • It is known for its readability and simplicity.
  • Python is widely used in web development, data analysis, artificial intelligence, scientific computing, and more.
  • Python code is typically saved with a .py extension.
  • In Python we can create variables, functions, classes ,lists, dictionaries, and more.

Hardware

  • Computer hardware refers to the physical components of a computer system.
  • Key hardware components include the CPU, RAM, hard drive, motherboard, and external intruments like keyboard and mouse.
  • Hardware can be categorized into input devices, output devices, storage devices, and processing units.
  • Input devices include keyboards, mice, and scanners.
  • Output devices include monitors, printers, and speakers.
  • Storage devices include hard drives, SSDs, and USB flash drives.
  • Processing units include the CPU and GPU.
  • Hardware can be upgraded or replaced to improve performance.

Internet

  • The Internet is a global network of interconnected computers.
  • It allows for the sharing of information and communication between users worldwide.
  • Key components of the Internet include web browsers, servers, and protocols like HTTP and FTP.
  • Web browsers are software applications used to access and view websites.
  • Servers host websites and provide resources to users.
  • Protocols define the rules for data transmission over the Internet.
  • The Internet supports various services, including email, social media, online gaming, etc.
  • Security measures like firewalls and encryption are essential for safe online communication.

Multimedia

  • Multimedia refers to the integration of text, images, audio, and video content.
  • It enhances the user experience by providing interactive and engaging content.
  • Common multimedia formats include JPEG, PNG, MP3, MP4, and GIF.
  • Multimedia can be used in presentations, websites, games, and educational materials.
  • Tools for creating multimedia content include graphic design software, video editing software, and audio production software.
  • Multimedia content can be shared online through platforms like YouTube, Vimeo, and social media.

Excel

  • Excel is a spreadsheet program developed by Microsoft.
  • It is used for data organization, analysis, and visualization.
  • Excel allows users to create tables, perform calculations, and generate charts.
  • Key features include formulas, functions, creating tables, and conditional formatting.
  • Excel files are typically saved with a .xlsx or .xls extension.
  • Excel is widely used in business, finance, and academic settings for data management.
  • Excel can be used for tasks like budgeting, forecasting, and data analysis.
